MINI PROJECT 1

Roguelike Dungeon

Deskripsi Project

Project ini adalah sebuah game Roguelike Dungeon berbasis browser yang sepenuhnya dibuat menggunakan HTML, CSS, dan JavaScript murni. Pemain mengendalikan karakter yang menjelajahi dungeon yang di-generate secara acak, melawan musuh, mengumpulkan gold, dan naik level semakin dalam ke dungeon. Game ini memiliki sistem yang cukup lengkap layaknya game roguelike sungguhan.

Fitur Yang Ada

Game ini dilengkapi dengan berbagai fitur, di antaranya: sistem HP, Level, dan XP yang bertambah saat mengalahkan musuh; sistem Gold untuk bertransaksi dengan merchant; indikator Depth (kedalaman dungeon) dan Turns (jumlah langkah); fitur Inventory dengan slot item 1-5 yang bisa digunakan lewat hotkey; serta fitur Save & Load untuk menyimpan dan melanjutkan permainan. Terdapat juga NPC Merchant yang muncul di dungeon dan bisa diajak berinteraksi untuk berdagang.

Teknologi Yang Digunakan

  • HTML
  • CSS
  • JavaScript
  • DOM ManipulationLocal
  • LocalStorage (Save/Load)
  • Keyboard Event Listener
  • addEventListener

Cara Bermain

Gunakan tombol navigasi panah keyboard atau tombol WASD untuk menggerakkan ikon karakter menjelajahi lorong labirin hitam. Dekati ikon musuh untuk memulai serangan otomatis berbasis giliran, kumpulkan peti emas, dan tekan tombol angka 1-5 pada keyboard untuk mengonsumsi ramuan pemulih HP dari tas inventory.

Hal Yang Saya Pelajari

Melalui project ini, saya belajar cara memanipulasi DOM secara intensif menggunakan JavaScript untuk membuat game yang interaktif dan real-time di browser. Saya juga belajar menggunakan addEventListener untuk menangkap input keyboard, logika game loop, serta localStorage untuk menyimpan data game secara permanen. Project ini mengajarkan saya bahwa JavaScript tidak hanya untuk website biasa, tetapi juga bisa digunakan untuk membuat game yang sesungguhnya.
