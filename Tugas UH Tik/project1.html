HALAMAN BIODATA SISWA

Biodata Siswa

TAMPILAN PROJECT

Deskripsi Project

Project ini merupakan halaman portofolio personal yang menyajikan informasi data diri, karakter pribadi, serta minat/hobi ke dalam susunan tiga kolom yang proporsional. Dikembangkan menggunakan kombinasi HTML dan Bootstrap 5, halaman ini memaksimalkan fitur grid system bawaan framework agar seluruh komponen visualnya otomatis adaptif di berbagai ukuran layar

Fitur Yang Ada

Halaman ini terdiri dari tiga kolom utama: Biodata (nama, tempat lahir, kelas), Kepribadian (deskripsi singkat diri), dan Hobi (daftar hobi dalam bentuk list). Di bagian bawah terdapat tiga tombol: Like, Post, dan Back yang dibuat menggunakan komponen Button Group dari Bootstrap.

Teknologi Yang Digunakan

  • HTML
  • Bootstrap
  • CSS
  • Grid System

Hal Yang Saya Pelajari

Melalui project ini, saya belajar mengoptimalkan fitur grid dari Bootstrap 5 untuk menyusun halaman web multi-kolom yang responsif secara lebih efisien. Selain menghemat waktu karena tidak perlu menulis banyak CSS dari nol, saya juga mendapat pengalaman dalam menerapkan utilitas siap pakai seperti Button Group, fungsionalitas border, serta pemanfaatan class spacing untuk mengatur jarak antar-komponen.
