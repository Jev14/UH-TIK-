Deskripsi Project

Project ini adalah sebuah website toko online produk Rice Cooker merek Cosmos yang dibuat menggunakan HTML, CSS, Bootstrap 5, dan JavaScript. Website ini mensimulasikan halaman promosi produk lengkap dengan katalog produk, halaman detail tiap produk, form pemesanan interaktif, dan fitur order via WhatsApp langsung dari browser.

Fitur Yang Ada

  • Banner Promosi Hero: Tampilan visual penawaran produk Rice Cooker Cosmos dengan tombol aksi belanja langsung.
  • Katalog Detail Produk: Penyusunan varian ukuran rice cooker lengkap dengan spesifikasi watt, keunggulan anti-lengket, dan harga resmi.
  • Form Pemesanan Interaktif: Input data pembeli (Nama, Alamat, Jumlah Barang) yang langsung divalidasi sistem agar tidak kosong.

Teknologi Yang Digunakan

  • HTML
  • CSS
  • JavaScript Data
  • Bootstrap 5
  • WhatsApp API
  • Animate.css
  • Multi-page Navigation

Hal Yang Saya Pelajari

Melalui project ini, saya belajar membangun situs web multi-halaman yang saling terhubung dan memanfaatkan komponen Bootstrap 5 untuk mempercepat proses pembuatan desain toko online yang tampak profesional. Selain itu, saya menerapkan logika JavaScript agar sistem dapat menghitung total belanjaan pembeli secara langsung saat itu juga. Pengalaman berkesan saya saat membuat project ini adalah mengintegrasikan formulir order dengan WhatsApp API melalui tautan wa.me, sehingga nota belanja bisa langsung terkirim otomatis ke chat WhatsApp dan membuat website ini berfungsi penuh seperti toko online sungguhan.
